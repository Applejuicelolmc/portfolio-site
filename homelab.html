Juicelab

I've started this project around 2020 when I bought a second hand Lenovo m715q tiny to host my own discord bot
I decided to install Proxmox on this machine, then I installed an ubuntu VM and made a simple startup script that runs when the VM boots.

Along the way I came along docker and portainer to run services such as pihole to block certain dns requests. I really liked the ease of use of docker so I made my bot available on docker and started hosting it via docker instead of a VM.

This homelab has now evolved to having 4 Lenovo m715q Tiny's with upgrades to 2.5GB NICS in a cluster, a Terramaster F424 Pro NAS with 2 14TB drives mirrored for backup and file storage.

For the future I'm planning to build a router from a Lenovo m920Q Tiny and to use some of my raspberry pi's that are just laying around for other projects.

Shrimp bot

I've started this project by the end of 2018, at the time I really wanted to have a custom discord bot so I decided to learn how to program with javascript by making this bot. The bot has commands that consist mostly of small games such as rock paper scissors.
I was still in high school studying an unrelated field so I had to learn everything from scratch. Somewhere around 2019-2020 I decided to switch from javascript to typescript since they were pretty similar so te learning curve wasn't too great.
